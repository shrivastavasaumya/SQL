{ Term }                          -->      { Brief Explanation }

 Database                          :  An organized collection of related data stored electronically. 

 DBMS (Database Management System) :  Software application used to create, manage, update, and retrieve data from databases. 

 SQL (Structured Query Language)   :  A language used to communicate with and manipulate data in a database.
